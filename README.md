This is a **Care.xyz** project built with [Next.js](https://nextjs.org) using [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## About the Project

**Care.xyz** is a caregiving service platform where users can easily find and book trusted caretakers for:

- Baby Sitting
- Elderly Care
- Sick People Care

The goal of this platform is to make caregiving **easy, secure, and accessible for everyone**.

Users can select their required service, choose duration and location, and book the service easily through the platform.

---

## Features

- Responsive Design (Mobile, Tablet, Desktop)
- Email & Password Authentication
- Google Social Login
- Dynamic Booking System
- Automatic Total Cost Calculation
- Booking Status Tracking
- My Bookings Dashboard
- Service Detail Pages
- Email Invoice System (Booking Confirmation)

---

## Services

The platform currently provides the following services:

- **Baby Sitting Service**
- **Elderly Care Service**
- **Sick People Care Service**

Each service has a detailed page where users can learn more and book the service.
